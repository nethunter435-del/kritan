Hi — I'm Kritan, a Class 12 Science student from Nepal 🇳🇵. I'm deeply passionate about cybersecurity, and my main focus is ethical hacking and red team operations.

My journey into hacking started out of pure curiosity — I wanted to understand how things break before I could understand how they work. I'm self-teaching using free platforms like TryHackMe, HackTheBox, and YouTube, alongside my school studies.

My current skill set spans Python scripting, C programming, computer networking, basic frontend web development, and Linux systems. I spend hours on CTF (Capture the Flag) challenges and reading documentation most students my age wouldn't touch.

I built this entire website myself — every line of HTML, CSS, and JavaScript written by hand. My goal was simple: create the resource I wish I had when I started. Free, deep, honest, and built for students like me — from Nepal and everywhere else in the world.

I believe that the best hackers come from places people least expect. Nepal is not Silicon Valley — but the curiosity here is just as sharp, and the hunger to learn is just as real.

Cybersecurity knowledge is hidden behind expensive courses and paywalls. That's wrong. A student in Kathmandu deserves the same quality education as someone in San Francisco. This site is my contribution to changing that — completely free, forever.

// 04. LEARNING PATH

Complete Roadmap

The exact path from zero to ethical hacker — with free resource links for every step.

01

NETWORKING FUNDAMENTALS

You cannot hack networks you don't understand. Learn OSI model, TCP/IP, DNS, HTTP/HTTPS, IP addressing, subnetting, ports, and protocols. This is the most important foundation.

OSITCP/IPDNSSubnettingWireshark
NetworkChuck YT Prof. Messer Network+ Wireshark Docs
02

LINUX MASTERY (KALI)

Linux is the hacker's home. Learn navigation, file permissions, bash scripting, process management, cron, grep, awk, netcat, and setting up Kali Linux as your primary pentest OS.

Kali LinuxBashPermissionsScripting
OverTheWire: Bandit Kali Docs Linux Journey
03

PYTHON FOR HACKERS

Learn Python to build your own tools. Write port scanners, password crackers, reverse shells, and automation scripts. Understanding code makes you a far better hacker.

Python3SocketsScriptingAutomation
TCM Security: Python Harvard CS50P (Free) Python Official Docs
04

RECONNAISSANCE & OSINT

Information is everything. Learn passive and active recon — Google dorking, Shodan, Maltego, theHarvester, Recon-ng, WHOIS lookups, and building target intelligence profiles.

OSINTShodanGoogle DorksMaltego
Shodan.io OSINT Framework THM: Google Dorking
05

SCANNING & ENUMERATION

Discover live hosts, open ports, service versions, and OS details. Master Nmap inside out — stealth scans, NSE scripts, firewall evasion. Learn SMB, FTP, SNMP enumeration.

NmapGobusterNiktoEnum4linux
Nmap Manual THM: Nmap HackTricks: SMB
06

WEB APPLICATION HACKING

OWASP Top 10 is your bible. SQL Injection, XSS, CSRF, IDOR, SSRF, file uploads, JWT attacks. Use Burp Suite to intercept and modify every request. Practice on DVWA and PortSwigger labs.

OWASP Top 10SQLiXSSBurp SuiteIDOR
PortSwigger Academy OWASP Top 10 DVWA
07

EXPLOITATION

Turn discovered vulnerabilities into access. Learn Metasploit framework, manual exploitation, CVE research, payload generation with msfvenom, and setting up listeners for reverse shells.

MetasploitMsfvenomShellsCVEs
Metasploit Unleashed Exploit-DB THM: Metasploit
08

PRIVILEGE ESCALATION

Getting in is only half the job. Learn Linux (SUID, cron, writable files, sudo misconfigs) and Windows (token impersonation, DLL hijacking, unquoted service paths) privesc techniques.

Linux PrivEscWindows PrivEscGTFOBinsLOLBAS
GTFOBins LOLBAS THM: Linux PrivEsc
09

CTF COMPETITIONS & PRACTICE

Apply everything in real-time competitions. CTFs sharpen your thinking, expose you to new techniques, and are the best way to build a portfolio. Start with PicoCTF then move to HTB/THM.

PicoCTFHTBTryHackMeCTFtime
PicoCTF HackTheBox CTFtime.org

// 08. CODE OF ETHICS

The Ethical Pledge

My personal code — every learner on this site is expected to honor this.

I'm Kritan — a student, not a criminal. Everything I learn here is for defense, education, and authorized testing only. The skills are powerful; using them wrong destroys trust and lands you in prison. Use them right and you protect the world.

  1. 01
    I will never access any system without explicit written permission from the owner. No exceptions.
  2. 02
    I will define scope clearly before every engagement and never exceed those boundaries.
  3. 03
    I will report all vulnerabilities responsibly to authorized parties. I will never exploit them for personal gain.
  4. 04
    I will protect confidential data I encounter during any authorized test. I will not retain or share it.
  5. 05
    I will practice only on systems I own or have explicit permission to test — labs, VMs, CTF platforms.
  6. 06
    I will continuously learn and improve my skills ethically through legal channels only.
  7. 07
    I will use this knowledge to protect, not to harm. Cybersecurity exists to defend people.
  8. 08
    I will share free knowledge generously — just like this site does — because education should have no gatekeepers.
// — Kritan, Nepal 🇳🇵
